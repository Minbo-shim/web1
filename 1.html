HTML이란 무엇인가?

Hypertext Markup Language(HTML) is standard markup language for creating web pages and web applications Web browsers receive HTML documents from a server or from local storage and render them into multimedia web pages.

HTML describes the structure of a web page semantically and originally included cues for appearance of the document. HTML elemenHts are the building blocks of HTML pages. With HTML contructs, images and other objects, such as interactive forms, may be embedded into rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes, and other items. HTML elements are delineated by tags, written using angle brackets
